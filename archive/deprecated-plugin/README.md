# @forge/plugin — Framer Plugin

## What it is
Framer CMS plugin that embeds Forge-managed data into Framer sites.

## Why it exists
Allows Framer users to display dynamic data from Forge projects without custom backend code.

## Dependencies
- @forge/sdk — client SDK for API communication
- @forge/types — shared type definitions

## Depended on by
- Framer site editors / end-users
