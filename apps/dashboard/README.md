# @forge/dashboard — Admin Dashboard

## What it is
Next.js admin dashboard for managing the Forge BaaS platform. Provides UI for users, projects, and settings.

## Why it exists
Centralized management interface for platform administrators.

## Dependencies
- @forge/ui — shared UI components (shadcn/ui)
- @forge/sdk — client SDK for API communication
- @forge/types — shared type definitions

## Depended on by
- Internal administrators
