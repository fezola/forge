# @forge/sdk — Client SDK

## What it is
TypeScript client SDK for consuming the Forge API. Provides a clean, typed interface for all API endpoints.

## Why it exists
Abstracts HTTP communication with the Forge API. Used by the dashboard, plugin, and external consumers.

## Dependencies
- @forge/types — shared type definitions

## Depended on by
- apps/dashboard
- apps/plugin
- External consumers (published to npm)
